
注意事項
・｢$base｣フォルダは､SDカードの一番最初の所に配置して下さい｡
・全て自己責任の下､作業して下さい｡
・systemパーティション全体をバックアップしておく事をお勧めします｡


解説
$base/
・APKs/：Google等のシステムアプリのファイル => /system/app/
・APKs/Tools/：この作業を行う上で必要となって来るアプリファイル
・APKs/Update/：強制アプデやPlayストアが自動DLされない場合に必要なアプリファイル
・Permissions/：システム権限のファイル => /system/etc/permissions/
・Frameworks/：フレームワーク => /system/frameworks/
・bootanimation.zip => /system/media/
起動する時のアニメーション


参考
・KingRoot (is.gd/cpadroot)
端末でroot権限を取得するのに必要なアプリ
・ファイルマネージャー (is.gd/cpadex)
SDカードの中身を表示するのに必要なアプリ
